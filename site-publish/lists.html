Apache Pivot Mailing Lists

The following mailing lists relate to the Apache Pivot project. Users should consider subscribing to the User Support mailing list. The Developer list is for people interested in helping with the development and debugging of the toolkit.

Formatted archives are available at the Apache Mail Archives.

  • User support and discussion
  • Development mailing list
  • Source change reports


user

The list listed here is the only place that configuration and how-to questions belong, ever. Every day, users become frustrated, not with Apache Pivot, but how nobody will help them understand how to get started. 95% of the time, it's not for lack of helpers, but for lack of effort and well stated questions.

Before you go further, please read Eric S. Raymond and Rick Moen's explanation of "How To Ask Questions The Smart Way". If you follow their advice, you will discover that these forums have dozens of helpers ready to provide you with guidance on using Apache Pivot! Note: Do not send your Apache Pivot questions to Eric or Rick themselves!

Volume: Low
Subscription: Send a blank email to user-subscribe@pivot.apache.org
Digest subscription: Send a blank email to user-digest-subscribe@pivot.apache.org
Posting questions to the list: Send your question to user@pivot.apache.org (Replies are sent to the list, so you must be subscribed to see the answer!)
Getting help with the list: Send a blank email to user-help@pivot.apache.org
Unsubscription: Send a blank email to user-unsubscribe@pivot.apache.org
Digest unsubscription: Send a blank email to user-digest-unsubscribe@pivot.apache.org
List owner address: Send your message to user-owner@pivot.apache.org
Archives: Apache.


dev

The dev@pivot.apache.org mailing list is used for discussions about the actual development of the toolkit. It is not the place to ask about problems writing Pivot applications.

Configuration and support questions should be addressed to the user mailing list. This development list is only for discussion of changes to the Apache Pivot source code and related issues. Other questions will be ignored.
Volume: High
Subscription: Send a blank email to dev-subscribe@pivot.apache.org
Digest subscription: Send a blank email to dev-digest-subscribe@pivot.apache.org
Posting questions to the list: Send your question to dev@pivot.apache.org (Replies are sent to the list, so you must be subscribed to see the answer!)
Getting help with the list: Send a blank email to dev-help@pivot.apache.org, dev-digest-help@pivot.apache.org
Unsubscription: Send a blank email to dev-unsubscribe@pivot.apache.org
Digest unsubscription: Send a blank email to dev-digest-unsubscribe@pivot.apache.org
List owner address: Send your message to dev-owner@pivot.apache.org
Archives: Apache.


commits

The commits@pivot.apache.org mailing list is used for notifications of changes that have been committed to the SVN repository. It is not for discussions.

Subscription: Send a blank email to commits-subscribe@pivot.apache.org (Not a user support list!)

Unsubscription: Send a blank email to commits-unsubscribe@pivot.apache.org
